SPIS Instruction Set Architecture

8-bit data bus
12-bit address bus

Instructions

0 - ADD A, B
1 - SUB A, B
2 - ADC A, B
3 - SBB A, B
4 - SWP A, B
5 - LOAD A, imm8
6 - LOAD A, [disp12]
7 - LOAD A, [disp12+C]
8 - STORE [disp12], A
9 - STORE [disp12+C], A
A - SWP A, C
B - JE disp12
C - JMP disp12
D - NOT A
E - AND A, B
F - OR A, B

Memory Map

000 => BFF |     3 kiB | Code ROM
C00 => EFF | 768 bytes | RAM
F00 => FFF | 256 bytes | Display Output (32x8 ASCII)

"Hello, world!" example (40 bytes)

main:
  LOAD A, 0
  SWP A, C
  LOAD A, message.length
  SWP A, B

.loop:
  LOAD A, [message+C]
  STORE [0xF00+C], A

  ; Increment message pointer
  SWP A, C
  SWP A, B ; These 3 SWP instructions are equivelent to a "SWP B, C" (which doesn't exist)
  SWP A, C
  LOAD A, 1
  ADD A, B

  ; Check if End-of-Message has been reached
  SWP A, B
  SWP A, C
  CMP A, B

  ; Make registers the correct order
  SWP A, B
  SWP A, C

  ; If End-of-Message was reached jump to end, otherwise jump to start of loop
  JE .end
  JMP .loop
.end:
  JMP $

message:
  #d "Hello, world!"
.length = $ - .message
